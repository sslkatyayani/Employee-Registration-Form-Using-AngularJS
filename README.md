# Employee-Registration-Form-Using-AngularJS
A dynamic and responsive Employee Registration Form built using AngularJS with client-side validation and Excel export functionality. This project demonstrates form handling, validation, and data processing in a single-page web application.

## Project Overview

This project provides a professional employee registration system with:
- Real-time form validation
- Clean UI with a hero banner
- AngularJS two-way data binding
- Option to download submitted data as an Excel file
- It is suitable for college mini projects, web development practice, and frontend demonstrations.

## Technologies Used

- HTML5
- CSS3
- AngularJS (1.8.2)
- JavaScript

## Features

- Dynamic form handling using AngularJS
- Client-side validation for all input fields
- Email and phone number format validation
- Department selection using dropdown
- Responsive form layout with modern UI
- Excel file download functionality
- Clean and structured codebase

(No server setup required)

## Form Validations Included

- Full Name – Required
- Email – Required and must be valid
- Phone Number – Must be a 10-digit number
- Department – Mandatory selection
